## 安装Java
1. 下载安装
 1. 下载
		
 2. 解压
		tar -xvzf 解压的文件路径+文件名 （-C 解压的目的文件夹）

 3. 修改配置文件（环境变量）
		vim /etc/profile
		（vi ~/.bash_profile  用户）
		
		添加在文件末尾
		#java setting
		export JAVA_HOME=<你的Java解压路径>
		export JRE_HOME=${JAVA_HOME}/jre 
		export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib 
		export PATH=${JAVA_HOME}/bin:$PATH
		
	
 4. 重新加载配置文件
		source /etc/profile
		
 5. 验证
		java -version
	
2. 命令安装
	yum install java-1.8.0*
